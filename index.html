Breve Introducción a HTML

HTML (Hypertext Markup Language) es un lenguaje de marcado de etiquetas, que es utilizado para definir la estructura que debe seguir el contenido de una página web. HTML consiste es una serie de etiquetas que se utilizan para envolver diferentes partes del contenido para hacer que se muestre en pantalla de una manera concreta o que actúe de una manera concreta.
